## Особенности датасета

1. **Синтетический** - генерируется из русских шрифтов
2. **Аугментирован** - каждое изображение имеет вариации:
   - Шум
   - Размытие
   - Искажение
   - Волнистые трансформации
   - Возрастающий эффект
   - Бинаризация

3. **Типизированный фон**:
   - Белый 30%
   - Кремовый 25%
   - Линованный 20%
   - Сетка 15%
   - Состаренный 10%

4. **Оптимизирован для M1** - быстрая генерация (16 процессов)

## Что изменилось в train.py

### Было:
```python
# Все классы = все символы из всех меток
for record in all_records:  # включая слова/предложения!
    for char in record["label"]:
        unique_chars.add(char)
# Проблема: класс "привет" → классы 'п', 'р', 'и', 'в', 'е', 'т'
```

### Стало:
```python
# Разделяем: символы vs слова/предложения
symbol_records, word_records, sentence_records = step1_scan_datasets()

# Классы только из букв/цифр/знаков
for record in symbol_records:  # только Letters/Nums/Punkt!
    first_char = record["label"][0]
    unique_chars.add(first_char)
# Теперь: правильное разделение на классы
```

### Правильное использование:
- **symbol_records** → CNN тренировка
- **word_records** → валидация (опционально)
- **sentence_records** → финальный тест
